圣杯布局与双飞翼布局

发表于 2016-07-03   |   分类于 css   |  

圣杯布局和双飞翼布局：左右两栏固定宽度，中间栏main自适应的三栏布局；而且中间栏main要放在其余两栏的前面优先渲染。

圣杯布局

圣杯布局的原理及步骤

1.三栏浮动float:left；
2.主块设置width:100%占满宽度；
3.左块使用margin-left:-100%拉到最左边，右块使用margin-left: -rw px，rw为右块的宽度；
4.父容器设置左右padding容纳下左右块；
5.左块用相对定位拉到父容器在左边空出的位置，右块同理；

双飞翼布局

双飞翼布局原理及步骤

1.主块添加一个容器；左右块和主块容器main-wrap浮动float:left；
2.主块容器main-wrap设置width:100%；
3.左块使用margin-left:-100%拉到最左边，右块使用margin-left: rw px，rw为右块的宽度；
4.设置主块左右margin；

圣杯布局和双飞翼的比较

  • 相同：
    1.两种布局都是把主要内容放在前面加载；
    2.布局有相似之处，都使用了负外边距来布局；

  • 不同：
    圣杯布局是设置 父容器的padding值、相对定位来实现；
    双飞翼布局是给main添加了一个div容器、设置main的左右外边距 来实现，没有使用相对定位；
