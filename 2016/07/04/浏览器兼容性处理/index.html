CSS hack

针对不同的浏览器/不同版本写出相应的CSS代码，叫做CSS hack
原因：不同厂商的流览器或某浏览器的不同版本，对CSS的支持、解析不一样，导致在不同浏览器的环境中呈现出不一致的页面展现效果

CSS Hack的3种表现形式

  • IE条件注释法（HTML头部引用if IE；IE专有的hack方式，Microsoft官方推荐）

  • CSS属性前缀法
    _和-对ie6有效；
    *对ie6,7有效；
    \9对ie11以下所有ie有效；
    \0对ie8,9,10有效；
    \9\0只对ie9,10有效；
    \0\9只对ie8,9有效；
    *+一般认为是IE7特有的hack

  • 选择器前缀法
    *前缀只对IE6有效；
    *+前缀只对IE7有效；

使用工具 browserhacks

  • 渐进增强和优雅降级

渐进增强：项目一开始就考虑低版本浏览器的兼容性，保证最基本的功能，再对高级浏览器进行效果改进。
优雅降级：一开始就构建完整的功能，完成后再进行低版本浏览器兼容；
